# 마석고 교복지도 — 사다리는 그만, 직접 배달!

아침 조회 시간, 부장 **김준식**이 3층을 직접 돌며 11개 학급(2-1 ~ 2-11)의
교복지도 담당 선생님을 배정하는 **3D 1인칭 탐험 게임**입니다.
사다리타기 대신 만들었습니다. 매번 배정이 무작위로 바뀌며,
**자기 반은 절대 배정되지 않습니다** (완전순열 보장).

## 🎮 플레이

- **웹에서 바로**: GitHub Pages 주소로 접속 (저장소 Settings → Pages 활성화 후)
- **로컬에서**: 폴더의 `시작.bat` 더블클릭 → 브라우저가 자동으로 열립니다
  (ES 모듈 때문에 로컬 서버가 필요해서 index.html 직접 더블클릭으로는 실행되지 않습니다)

## 🕹 조작

| 키 | 동작 |
|---|---|
| W A S D | 이동 |
| Shift | 달리기 (오래 달리면 쉰 살 무릎이…) |
| 마우스 | 시점 (캔버스 클릭으로 고정) |
| H / M | 도움말 / 음소거 |

## ✨ 구경거리

- 11개 교실 + 교무실/특별실/옥상까지 3층 평면도 기반 학교
- 교실 입장 시 "차렷, 경례!" → 담당 선생님과 비주얼노벨식 대화 (선생님별 개그 대사 66종)
- 복도 순찰 중인 교장·교감 선생님과 RPG식 조우
- 후드티 위반 학생 추격 단속, 지각생, 옥상에서 몰래 빵 먹는 학생들
- 학생 잡담 말풍선, 교내방송, 우하단 레이더, 완주 칭호까지

## 🛠 기술

Three.js (CDN import map) · 순수 JS ES 모듈 · Web Audio 합성 사운드 · 빌드 단계 없음
